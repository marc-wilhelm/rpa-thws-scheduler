THWS Scheduler

Business School

Vorlesungsplan importieren/aktualisieren
Stellt eine Anfrage an die API, welche alle angegeneb Vorlesungpläne nach Modulen absucht (scrapt) und anschließend als JSON zurückgibt. Über diese JSON Struktur, wird dann iterriert, um alle Tage und Vorlesungen zu bekommen, wobei anschließend für jede vorlesung ein Termin im Kalender neu eingetragen wird oder aktualisiert wird.
Kalender leeren
Löscht alle vorhanden Termine aus deinem Kalender.

API Info

URL:
https://thws-scheduler.azurewebsites.net/api/schedule_scraper

Content-Request:

{
  "plans": [
    {
      "url": "https://business.thws.de/fileadmin/share/vlplan/BBA%206%20SS%2025.html",
      "label": "BBA 6",
      "filter": ["PBA1", "SPBA"]
    },
    {
      "url": "https://business.thws.de/fileadmin/share/vlplan/BBW%206%20SS%2025.html",
      "label": "BBW 6",
      "filter": ["SP WINF"]
    }
  ]
}
