THWS Scheduler

Business School

Vorlesungsplan importieren
Importiert Vorlesungspläne und fügt die Veranstaltungen automatisch in deinen Kalender ein.
Kalender leeren
Entfernt alle importierten Veranstaltungen aus deinem Kalender.

Vorlesungspläne auswählen

Einfache Eingabe
JSON-Format

API-Informationen

URL:
https://thws-scheduler.azurewebsites.net/api/schedule_scraper

Anfrage-Format:

{
  "plans": [
    {
      "url": "https://business.thws.de/fileadmin/share/vlplan/BBA%206%20SS%2025.html",
      "label": "BBA 6",
      "filter": ["PBA1", "SPBA"]
    },
    {
      "url": "https://business.thws.de/fileadmin/share/vlplan/BBW%206%20SS%2025.html",
      "label": "BBW 6",
      "filter": ["SP WINF"]
    }
  ]
}

Hilfe und Dokumentation:

Die API durchsucht die angegebenen Vorlesungspläne und extrahiert alle Veranstaltungen. Du kannst optional Module mit dem Filter-Parameter eingrenzen.

Fehlermeldungen werden hier im Status-Bereich angezeigt. Bitte stelle sicher, dass die URL-Adressen korrekt sind und auf gültige THWS-Vorlesungspläne verweisen.
